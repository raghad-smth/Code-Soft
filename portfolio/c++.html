AVL Tree Implementation for Item Management

Overview

This project implements an AVL Tree, a self-balancing binary search tree, to efficiently manage and organize items based on their names and prices. The program demonstrates advanced features, including the ability to traverse items using multiple properties and ensuring the tree remains balanced after each operation.

Key Features

  • Dynamic Insertion and Deletion: Efficiently add and remove items while maintaining the AVL Tree balance.
  • Multi-Property Traversal: Traverse and display items in ascending and descending order based on either name or price.
  • Self-Balancing Mechanism: Automatic balancing through rotations to ensure optimal search, insertion, and deletion times.

Technologies Used

  • C++: Implementation of the AVL Tree and item management.
  • Object-Oriented Design: Use of classes and methods to encapsulate the functionalities of items and the AVL Tree.

    • Functionalities of the Program:

    1. Adding Nodes: The addNode method inserts new nodes into the tree while ensuring it remains balanced. This involves checking the balance factor and performing rotations if necessary.
    2. Deleting Nodes: The deleteNode method removes nodes from the tree, ensuring the tree remains balanced after each deletion.
    3. Tree Traversal: Unlike a typical AVL tree that might traverse based on a single property, this implementation can traverse the items using multiple properties, specifically names and prices. This is achieved by implementing separate in-order and reverse in-order traversal methods for each property:
      • By Name: Methods diplayOrderdByNameAsc() and diplayOrderdByNameDesc() display the items in ascending and descending order by name, respectively.
      • By Price: Methods diplayOrderdByPricAsc() and diplayOrderdByPricDesc() display the items in ascending and descending order by price, respectively.
    4. Balancing: The program maintains the balance of the tree by calculating the height and balance factor of each node, performing necessary rotations to keep the tree height-balanced.

    Implementation Details:

    • Item Class: Defines the properties and comparison operators for items, enabling sorting by name.
